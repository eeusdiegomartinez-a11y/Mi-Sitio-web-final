Preguntas Frecuentes

Aquí resolvemos las dudas más comunes para que aproveches al máximo todo lo que Progresso tiene para ti.

¿La app es gratuita? +

Sí, puedes usar todas las funciones básicas sin costo. Habrá opciones premium en el futuro.

¿Necesito estar conectado a internet? +

Solo para sincronizar los datos o recibir notificaciones. Puedes revisar tus tareas sin conexión.

¿Puedo usarla en varios dispositivos? +

Sí, puedes acceder desde tu celular o tableta con el mismo usuario.

¿Cómo se registran las tareas? +

Solo debes ingresar el nombre, materia, fecha y descripción. ¡Es muy fácil!

¿Es solo para estudiantes de colegio? +

Por ahora, sí. Progresso está pensada para la rutina académica escolar.
